# save-the-date

Eine einfache "Save the Date" Webseite mit Countdown und RSVP-Funktionalität.

## Funktionen

- Anzeige des Veranstaltungsdatums
- Live-Countdown bis zur Veranstaltung (Tage, Stunden, Minuten, Sekunden)
- RSVP-Button zum Zusagen
- Responsives Design

## Verwendung

1. Öffnen Sie `index.html` in einem Webbrowser
2. Die Seite zeigt automatisch den Countdown bis zum 21. Dezember 2025 an
3. Klicken Sie auf "Zusagen", um Ihre Teilnahme zu bestätigen

## Tests ausführen

Um die automatisierten Tests auszuführen:

1. Öffnen Sie `test.html` in einem Webbrowser
2. Die Tests werden automatisch ausgeführt und die Ergebnisse angezeigt

Die Test-Suite prüft:
- Korrekte Anzeige aller Seitenelemente
- Vorhandensein des Countdowns
- RSVP-Button-Funktionalität
- Bestätigungsnachricht nach RSVP

## Dateien

- `index.html` - Haupt-HTML-Datei
- `style.css` - Stylesheet für das Design
- `script.js` - JavaScript für Countdown und Interaktivität
- `test.html` - Automatisierte Tests für die Webseite
